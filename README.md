# Skills

Engineering agent skills for PR descriptions, breaking-change reviews, and safe SSH deployment operations.

## Install

```sh
npx skills add <owner>/<repo>
```

## List available skills

```sh
npx skills add <owner>/<repo> --list
```

## Use this checkout

```sh
git clone https://github.com/<owner>/<repo>.git
cd <repo>
npx skills add .
```

List local skills without installing:

```sh
npx skills add . --list
```
